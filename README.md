# Business Process Modeling – BPMN Diagrams (Public Sector)

This project consists of weekly business process modeling exercises using BPMN 2.0, based on real-world public sector data from the Greek government portal [mitos.gov.gr](https://www.mitos.gov.gr/).

## 🧭 Project Scope
- Modeled various administrative and public-facing processes (e.g., permit issuance, citizen requests, interdepartmental workflows)
- Used BPMN 2.0 standard for modeling consistency and clarity
- Visualized processes using Camunda Modeler

## 🛠 Tools Used
- Camunda Modeler
- BPMN 2.0 Notation
- Public sector case data from mitos.gov.gr

## 📁 Files Included
- `.bpmn` source files for each weekly exercise
- `.png` exports for quick visualization

## 📌 Sample Diagrams
- Public document request process
- Business license issuance
- Internal municipal approval workflow

## 🔍 Purpose
To develop and showcase applied BPMN modeling skills on real administrative scenarios, supporting digital transformation and process automation strategies.
